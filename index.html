AJAS

AJAS is static test pad which was developed as part of internship. STP are used to find the amount of thrust developed by the rocket motor.
Challenges I overcome in the production of STP are as follows
- Weight was reduced from 25 kgs to 3kgs by using Aluminum 80-20.
- Modular design allows testing motor in 2 differnt orientation.
- Precise reading were obtained with the help of adjustable base.
- Variable motor size can be achieved with the new motor clamp.
- Load is equally distributed to all the legs when load is applied.
- Max load = 350N and Max displacement = 0.1043mm.
